## Getting Started

### Prerequisites

Install Ionic

```
npm install -g cordova ionic
```

Install dependencies

```
npm install
```

### Run the app

Dev :

```
npm run ionic:serve
```

### Build

iOS :

```
npm run build:ios
```

Android :

```
npm run build:android
```
